title: shader基础语法

Unity Shader基础语法

POSITION 表示unity将模型顶点输入到顶点着色器v中，SV_POSITION表示顶点着色器输出到裁切空间中。
SV_Target将渲染目标输出到一个渲染器中

Run server

a2v:application to vert 应用阶段数据传入到顶点着色器中。
POSITION、TANGENT、NORMAL、TEXCOORD0、TEXCOORD1、TEXCOORD2、TEXCOORD3、TEXCOORD4
v2f:顶点着色器到片元着色器之间传递信息。

片元着色器中的输入，其实是顶点着色器的输出插值后的结果。

Color，Vertor– float4,half4,fixed4

1、函数ComputeScreenPos

ComputeScreenPos得到的结果是：齐次坐标系下的屏幕坐标值也就是x[0,w],y[0,w]
在tex2D()函数中我们可以这样使用，也就是将ComputeScreenPos的结果进行齐次化，取纹理值。

tex2D(_ScreenTex,float2(o.xy/o.w))

当然最常见的做法还是直接通过模型的纹理坐标进行取值
